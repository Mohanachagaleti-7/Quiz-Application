# Quiz-Application
A console-based Quiz Application developed using Python. The program stores questions using lists and dictionaries and calculates scores automatically with negative marking. It checks user input, controls quiz flow using loops and conditions, and calculates the final percentage with pass or fail result. The design allows easy addition of new question
